# FasoCare 🇧🇫 🏥✨

**Plateforme Nationale de Santé Numérique du Burkina Faso.**

FasoCare est une infrastructure numérique souveraine conçue pour digitaliser le parcours de soins, assurer la sécurité des données médicales et garantir l'accès universel aux services de santé, même en zones rurales via USSD et synchronisation hors-ligne.

## 📱 Écosystème FasoCare

- **FasoCare Mobile (React Native)** : L'application principale pour les citoyens et les professionnels de santé.
- **Supervision Nationale (Next.js)** : Tableau de bord de pilotage pour le Ministère de la Santé.
- **FasoCare Core (NestJS)** : API sécurisée avec RBAC, moteur QR et intégration USSD/SMS.
- **FasoCare Web Launcher** : Point d'entrée institutionnel et accès PWA pour ordinateurs.

## 🛡️ Architecture & Sécurité

- **Offline-First** : Synchronisation robuste pour les régions à faible connectivité.
- **Souveraineté des Données** : Isolation stricte des données PII sur le territoire national.
- **RBAC (Role Based Access Control)** : Accréditation granulaire pour Médecins, Pharmaciens et Administrateurs.

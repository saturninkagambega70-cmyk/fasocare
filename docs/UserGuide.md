# Guide Utilisateur - FasoCare

Bienvenue dans le guide de démarrage rapide de FasoCare, l'espace numérique de santé du Burkina Faso.

---

## 1. Pour les Citoyens et Parents

### 1.1 Inscription (Option Smartphone)
1. Téléchargez l'application FasoCare.
2. Saisissez votre numéro de téléphone (vérification par SMS ou USSD).
3. Lors de la connexion, définissez votre espace sécurisé.

### 1.2 Inscription (Option USSD - Téléphones Basiques)
1. Composez le `*xxx#`.
2. Suivez les instructions vocales ou textuelles pour créer votre NIP (Numéro d'Identification Patient).
3. Vous recevrez les rappels de rdv et alertes urgentes par SMS.

### 1.3 Gérer le carnet de son enfant
1. Allez dans l'onglet **Parent**.
2. Scannez le QR code fourni par la maternité ou ajoutez manuellement l'enfant.
3. Un rappel SMS ou Push vous alertera pour les prochains vaccins de votre enfant.

---

## 2. Pour les Médecins (CSPS, Cliniques, Hôpitaux)

### 2.1 Nouvelle Consultation
1. Connectez-vous avec votre identifiant `DOCTOR`.
2. Dans le **Dashboard**, cliquez sur "Scanner Patient" s'il a son code QR, ou recherchez-le par son numéro de téléphone.
3. Remplissez ses variables vitales (Signes Vitaux), puis passez à la case **Diagnostic**.

### 2.2 Délivrer une Ordonnance Sécurisée
1. À la fin de la consultation, validez le traitement.
2. Le système génère automatiquement un **Code QR Médical Inviolable**.
3. Le patient reçoit ce code directement sur son téléphone.

### 2.3 Signaler une Épidémie
1. Depuis l'accueil, si vous diagnostiquez 3 cas de maladie contagieuse, cliquez sur **Alerte Épidémique**.
2. Ces données remontent instantanément (en mode anonymisé) sur les écrans du Ministère.

---

## 3. Pour les Inspecteurs et le Ministère de la Santé

### 3.1 Utilisation du Dashboard
La vue "Admin" est exclusive aux hauts fonctionnaires.
1. Connectez-vous sur le portail Web de FasoCare avec votre rôle `INSPECTOR` ou `HEALTH_MINISTRY`.
2. Analysez les données consolidées : Taux de mortalité, taux de vaccination.
3. Exploitez la carte thermique ("Heatmap") pour anticiper les approvisionnements en vaccins.

### 3.2 Extraire des Rapports
1. Cliquez sur le bouton "Exporter".
2. Vous pouvez générer un **Rapport PDF hebdo/mensuel** ou exporter les données anonymisées au format **CSV** pour intégration dans vos propres systèmes d'analyse (Interopérabilité).
